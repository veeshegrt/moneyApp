## 🆚 Skill vs Agent

| | Skill (/test) | Agent (/test-automation) |
|---|---------------|--------------------------|
| **复杂度** | 简单 | 复杂 |
| **功能** | 运行测试 | 测试+分析+建议 |
| **速度** | 快 | 较慢 |
| **使用** | 日常开发 | 深度分析 |

### 选择建议

**用 Skill 当**：
- ✅ 快速运行测试
- ✅ 日常开发检查
- ✅ 不需要详细分析

**用 Agent 当**：
- ✅ 需要详细报告
- ✅ 分析失败原因
- ✅ 获取改进建议
- ✅ 质量审计
